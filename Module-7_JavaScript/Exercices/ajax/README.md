# Exercice AJAX — Section commentaires

Implémentation de fonctionnalités AJAX sur une section de commentaires connectée à une base de données MySQL.

## Fonctionnalités à implémenter

- **Voir plus** — charge les 10 commentaires suivants (requête GET)
- **Like** — incrémente le compteur de likes d'un commentaire (requête GET)
- **Commenter** — soumet un nouveau commentaire et l'ajoute à la liste (requête POST)

> Aucun rechargement de page ne doit avoir lieu.
