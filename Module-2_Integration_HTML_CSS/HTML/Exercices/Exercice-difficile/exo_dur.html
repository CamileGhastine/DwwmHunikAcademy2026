Introduction

Le web est construit sur trois piliers : HTML, CSS et JavaScript.

HTML - HyperText Markup Language

HTML est le langage de balisage pour créer des pages web.

Les balises HTML

Une balise est délimitée par des chevrons :

Balise Utilité Exemple
<p> Paragraphe <p>Du texte</p>
<img> Image <img src="...">
<a> Lien <a href="...">Lien</a>

Les balises orphelines

  • <br> - Retour à la ligne
  • <hr> - Ligne horizontale
  • <img> - Image
  • <input> - Champ de saisie
  • <meta> - Métadonnées

La structure d'une page HTML

Une page HTML respecte toujours cette hiérarchie :

  1. <!DOCTYPE html>
  2. <html>
    1. <head>
      • <meta>
      • <title>
    2. <body>
      • <header>
      • <nav>
      • <main>
      • <footer>
